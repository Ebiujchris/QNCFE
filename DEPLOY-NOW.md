## 📤 Deploy to Hostinger Now

### Step 1: Add Environment Variable on Hostinger
1. Log into Hostinger control panel
2. Go to your website settings
3. Find "Environment Variables" or "Advanced" section
4. Add this variable:
   ```
   Name: VITE_API_URL
   Value: https://qncbe.vercel.app/api
   ```

### Step 2: Upload Files
1. Go to File Manager in Hostinger
2. Navigate to `public_html` folder
3. Delete any old files (if this is an update)
4. Upload ALL files from the `dist` folder:
   - index.html
   - robots.txt
   - sitemap.xml
   - .htaccess
   - _redirects
   - assets/ folder (entire folder)
   - images/ folder (entire folder)
   - vite.svg

### Step 3: Verify Deployment
Visit these URLs to confirm everything works:

1. **Homepage**: https://qncsolutions.org/
2. **Robots.txt**: https://qncsolutions.org/robots.txt
3. **Sitemap**: https://qncsolutions.org/sitemap.xml
4. **About page**: https://qncsolutions.org/about
5. **Services**: https://qncsolutions.org/services
6. **Contact**: https://qncsolutions.org/contact

All pages should load without 404 errors!

### Step 4: Submit to Google Search Console
1. Go to: https://search.google.com/search-console
2. Click "Add Property"
3. Enter: `https://qncsolutions.org`
4. Choose verification method (HTML file upload is easiest)
5. Download verification file and upload to your site root
6. Click "Verify"
7. Once verified, go to "Sitemaps" section
8. Submit: `sitemap.xml`
9. Go to "URL Inspection"
10. Enter: `https://qncsolutions.org`
11. Click "Request Indexing"
